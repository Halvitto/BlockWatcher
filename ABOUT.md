# About BlockWatcher

BlockWatcher is a privacy-first, open-source macOS menu bar monitor for local
AI coding clients.

It reports real Claude and Codex quota windows where local provider data exists,
adds today's model and token activity from a bundled offline `ccusage` sidecar,
and detects other supported apps and CLIs without inventing percentages.

## Principles

- **Local first:** no account, API key, telemetry, or runtime network request.
- **Honest data:** only Claude and Codex display verified quota percentages.
- **Minimal output:** compact rows expand only when details are requested.
- **Curated detection:** supported clients have explicit commands, bundles,
  processes, data adapters, and logos.
- **Open development:** source, architecture, attribution, and tradeoffs are
  public.
